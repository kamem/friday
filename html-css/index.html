HTMLについて

HTMLはHyperText MarkupLanguageの略で、ウェブページを作成するための標準的なマークアップ言語です。 文章構造を表します。

CSSについて

CSSはCascading Style Sheetsの略で、HTMLの要素を装飾するためのスタイルシート言語です。
